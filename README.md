# AURA

Aplicação web moderna para análise e recomendações de produtos de cuidados com a pele, desenvolvida com Vue 3 e Vite.

## 🚀 Tecnologias

- **Vue 3** - Framework JavaScript progressivo
- **Vite** - Build tool rápida e moderna
- **Vue Router** - Roteamento para SPA
- **SCSS** - Pré-processador CSS
- **Iconify** - Biblioteca de ícones

## 📝 Características

- Interface moderna com design Frutiger Aero
- Análise de produtos com IA
- Sistema de recomendações personalizadas
- Modo claro/escuro
- Upload de foto de perfil
- Histórico de produtos utilizados
